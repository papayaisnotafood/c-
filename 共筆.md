#### switch
>```cpp
>if(d == 'a') ｛
>    // do sth
>}else if(d == 'b') {
>    // do another thing

>}else {
>    // do the other thing
>}
>```
>switch作法:
>```cpp
>switch(d) {
>    case 'a':
>        // do sth

>    case 'b':
>        // do another thing
>        break;
>    default:
>        // do the other thing
>｝
